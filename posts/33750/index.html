搜索广告与竞价广告

《计算广告》第五章笔记

1 搜索广告

量的保证不复存在，广告主需要自行调整效果与量的平衡；价格的约定也被去掉，每个广告主都可以随时调整关键词的出价

1.1 搜索广告产品形态

  • 搜索广告是以查询词为粒度进行受众定向，并按竞价方式售卖、按 CPC 结算的广告产品。

  • 搜索广告竞价的标的物是竞价关键词

  • 用户输入的查询与关键词相匹配，来决定是否触发某广告

  • 匹配可以精准匹配，也可以扩展，称为查询扩展

  • 搜索广告的产品和技术特点：

  1. 搜索广告的变现能力，即 eCPM 远远高于一般的展示广告，其产品原因是，用户主动输入的查询直接反映了用户的意图。
  2. 搜索广告的受众定向标签，就是上下文的搜索查询。由于搜索词非常强地表征着用户的意图，因此搜索广告可以进行非常精准的定向。相对于搜索词，根据用户历史行为得到的兴趣标签的重要性大打折扣，这一方面是因为其信号远不如搜索词强烈，另一方面是因为用户这样明确意图的任务是不能被打断的。因此，搜索广告里的eCPM由一般情形下的 r(a, u, c) 退化成了 r(a, c)。
  3. 搜索广告的展示形式与自然结果的展示形式非常接近，有原生广告的意味。对相关性的要求远远超过展示广告，因此在查询匹配广告时需要非常精细的策略和技术。
  4. 从搜索广告发展起来的竞价交易模式，已经逐渐发展为互联网广告最主流的交易模式。

1.2 搜索广告产品新形式

  • 搜索广告和合约广告与竞价广告网络都有区别。
  • 从产品发展的角度看，搜索广告有 3 个方面的探索趋势：
  1. 丰富文字链创意的展示形式，让其更富表现力，以提高点击率
    1. 在通用广告链接上增加更多有表现力的信息点 。与行业基本无关，可以规模化复制（2）
    2. 直接展示结构化的广告内容摘要，提供一些可直接访问的功能，减少用户跳转成本，提高推广效率（1）（为了照顾用户体验的一致性，往往不能完全采用搜索广告的竞价方式来运营，一般就某个行业进行阶段性的线下竞价）

    尚未成为搜索广告的主流，因为需要分行业设计产品和运营方案，规模化程度有限；另外，如果直接对接单品，会产生商品库对接等复杂的工程问题（个性化重定向也有此问题）

  2. 利用东区对相关性要求稍低的特点，设计一些拓展广告产品（不适用于移动搜索广告）
    1. 带有品牌意味的广告形式，提高用户对品牌的认知程度和后续黏性（3）
    2. 同类推荐，可以按照竞价广告的逻辑来运营。以接触潜在用户为目的。
  3. 优化广告与自然结果的关系，在保证相关性和广告效果的前提下提高收入（原生化探索）

1.3 搜索广告产品策略

在搜索广告中，排序依据 eCPM 可以表示成 r(a, c) = u(a, c) bid (a)，不过一般 u 会被质量度（quality score）替代，即还要考虑落地页质量等因素，避免广告主的恶意行为、促进市场长期发展。

  1. 查询扩展，常见的匹配方式有：
    1. 精确匹配：分词形成集合，查询集合 == 关键词集合。（{英语，培训} = {培训，英语}）
    2. 短语匹配：查询集合 >= 关键词集合。（{英语，培训，机构，哪个，好} > {培训，英语}）
    3. 广泛匹配：根据相关度，如「英语培训」与「外语学习班」、「四级证书」。可以获得较多流量，但精准性会打折扣
    4. 否定匹配：明确指出哪些词不匹配，灵活地关停一些低效流量
  2. 广告放置（ad placement）
  • 北区广告是黄金展示位置，搜索广告的绝大多数广告收入都来自北区广告
  • 北区广告平均条数称为 NFP（north foot print）或 ASN（average show number）
  • 考察新算法对 RPM 的影响时，如果 NFP 有显著变化，此时对比 RPM 没有太大意义
  • 考察一条广告能否进入北区，两个因素：相关性和 RPM
  • 北区一般会设定广告条数上限
  • 根据整体 NFP 的约束和收入的目标，可以通过数据模拟的方式确定相关性和 RPM 的最有阈值（13 章讨论）
  • 根据用户对广告的偏好、容忍程度，个性化的设置 NFP

1.4 产品案例

Google AdWords，淘宝直通车（在一些高商业价值的垂直搜索引擎（如电商、房产、汽车、应用下载）中，利用搜索广告的产品体系变现是需要最优先考虑的流量变现方式）

2 位置拍卖与机制设计

2.1 市场保留价

  • 为了控制广告的质量和保持一定的出售单价，竞价广告市场往往要设置一个赢得拍卖位置的最低价格，称为市场保留价（Market Reserve Price，MRP），俗称「起价」或「底价」
  • MRP 有两种设置方式：1. 对整个竞价市场采用同样的保留价格 2. 根据不同标的物的特性设置不同的关键词
  • 搜索广告中，一般会对北区单独设置一个较高的 MRP
  • 搜索广告中，一个特定关键词的最优底价仅与竞价和质量度的分布有关，通过拟合其被保留价阶段的分布为核心，可以计算出理论最优底价
  • 根据竞价广告主的 eCPM 分布，找到一个使填充率没有明显下降的 CPM 底价，然后根据质量度倒算其 CPC 底价
  • 由于考虑质量度，动态底价可以同时与 a, u, c 三个参数都有关系，做到完全动态

2.2 定价问题

  • 定价问题探讨的是，在一次位置拍卖中，给定各参与者的出价，及它们的预期收益，如何对最后获得某位置的广告主收取费用
  • 纳什均衡，对称纳什均衡
  • 广义第二高价（GSP）
    • CPM 结算，广义第二高价为下一位广告主的出价
    • CPC 结算，广义第二高价为 \(u_{s+1}b_{s+1}/u_s+\Delta\)
    • CPS 结算，广义第二高价为 \(u_{s+1}v_{s+1}b_{s+1}/u_{s}v_{s}+\Delta\)

    广义第二高价的情况下，对某广告主的定价不会不会大于其出价

  • VCG 定价
    • 该定价策略的稳定状态下，市场中的理性广告主是忠实出价的
    • 向广告主收取的费用最少
    • 单广告位拍卖时，退化为广义第二高价
    • 媒体收入不如 GSP
      • 逻辑复杂，难以向广告主解释
      • 难以验证媒体是否正确计算了「给其他市场参与者带来的价值损害」

2.3 价格挤压

  • \(r = u^k bid\)，k 称为挤压因子，k 越大，bid 作用越小，反则反之
  • 若市场上存在大量出价高但品质低的广告主，可以增大 k 以强调质量和用户反馈
  • 若市场的竞价激烈程度不够，可以调低 k 以鼓励竞争
  • 若存在短期财务压力，可以调低 k

2.4 Myerson 最优拍卖

  • 相比社会福利最优，广告平台更关注平台收益最优，称为最优拍卖问题
  • 当拍卖的标的时单个的不可分物品时，存在理论上最优机制，称为 Myerson 拍卖
  • Myerson 拍卖机制：
    1. 对广告主的出价做映射，变为虚拟出价：\(b’ = b - \frac{1 - F(b)}{f(b)}\)
    2. 以 0 作为虚拟出价上的 MRP，对出价者进行过滤
    3. 在虚拟出价上按第二高价进行定价，并转换回真实出价 > 不保证按照出价高低排序，有时甚至会拒绝所有出价者 > 当 f 为 [0, 1] 内均匀分布时，此机制等价于第二高价加上 0.5 的底价 > 对于期望收益可能很高的广告主，通过虚拟出价的变换对其进行惩罚，以刺激其出更高的价格（期望收益高的广告主的 b’ 会偏小，导致无法中标）

3 竞价广告网络

为了给媒体的剩余流量提供变现机会，产生了广告网络这一产品，其批量聚合各媒体的剩余流量，按照人群或上下文标签的流量切割方式售卖给广告主

3.1 广告网络产品形态

  • 广告网络既有竞价的售卖方式，也有合约的售卖方式
  • 售卖的标的主要是人群，广告位被淡化了
  • 当流量满足多个广告活动时，可采用竞价模式而不考虑量的约束
  • 广告网络产品的来源有 2 个：1. 由搜索引擎发展出来的上下文广告产品（根据页面关键词来投放广告） 2. 由精细化受众定向发展出的兴趣广告产品
  • 竞价广告网络的产品特点：
    1. 不向广告主做量的约定
    2. 很难拿到品牌溢价高的广告位，不适合品牌广告
    3. 提高了市场流动性
    4. 财务上采用广告主先充值的方式，使得广告网络运营方的现金流状况改善
  • 一般采用 CPC 结算
  • 某些广告网络会对一部分合作供给方开放广告库供其自行挑选，该运营模式称为联盟

3.2 广告网络产品策略

大致分为检索、排序、定价三个步骤

  1. 广告检索
  2. 广告排序：CPC 结算时，CTR 预估面临以下困难：
    1. 点击数据稀疏
    2. 需要同时考虑上下文和用户两方面，使得新广告、新策略的冷启动问题严重
    3. 由于广告位的差别巨大，点击率的变动范围很大，使得稳健地估计点击率变得困难

3.3 产品案例

两种不同的业务方向： - 水平广告网络：广泛承接各种广告库存，面向所有品类的广告主进行服务 - 垂直广告网络：专门服务某一类型的广告主（如电商、游戏等），并寻找相关的媒体资源

4 竞价广告需求方产品

需求方的技术需求： 1. 按照广告主预算，跨媒体一站式采买人群（如何挑选合适的人群） 2. 机器决策的 ROI 优化（如何对各个目标人群给出合适的出价）

4.1 搜索引擎营销

  • 搜索引擎营销（SEM）：通过竞价采买搜索引擎关键词来做推广（关键词选择 + 出价）
    1. 关键词选择：广告主更信赖代表需求方利益的 SEM 公司
    2. 出价：广告主根据到达率、转化率、客单价来估计点击价值，以此参考来出价。但各关键词有很大不同，需要对不同关键词做不同出价

4.2 交易终端

  • 交易终端（TD）：即面向 ADN、DSP 等的一站式采买平台
  • 技术挑战主要在于 ROI 的优化部分。需要合理选择受众定向条件，并在每个人群上优化出价，以保证量和优化收益，这是个复杂的组合优化问题
  • 采取 API 接入的形式

4.3 产品案例

  • 对于非实时的竞价广告产品，需求方的优化比较困难，因此投资回报比较高的 SEM 得到了充分发展
  • Efficient Frontier

5 竞价广告与合约广告的比较

  • 从供给方来看，合约广告与竞价广告分别类似于计划经济和市场经济
    • 合约广告：所有量的保证和质的优化都由媒体方完成
    • 竞价广告：媒体方只负责制定竞价和收费规则，广告主量的保证完全由市场竞争的方式来完成
  • 竞价广告对广告市场的影响：
  1. 非常精细的受众定向被用在交易中，提升了市场效率
  2. 大量中小广告主称为竞价交易的主体，扩大了市场规模
  3. 数据的价值得以彰显，广告市场开始以数据为核心来组织和运营广告产品
